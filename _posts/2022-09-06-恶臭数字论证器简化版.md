# 恶臭数字论证器

&emsp; 如果不知道恶臭数字论证器是什么的，具体情况可以参考这个网站：[恶臭数字论证器](https://lab.magiconch.com/homo/)

&emsp; 现在我们要做的就是手写一个跟这个差不多的恶臭数字论证器。

## 原理

&emsp; 众所周知，一个数字 $x$ 可以拆成每一个数位上的数字 $x0, x2, x3, \cdots, x_n$，具体举个例子，就是 $123$ 可以拆成 $1, 2, 3$。并且我们可以很轻松的用 $\{x_i\}$ 这个数列表示出 $x$ 这个数，也就是：

$$ x = \sum_{i = 0}^n x_i10^i $$

&emsp; 这里观察一下，我们发现一下性质，也就是 $x_i \leq 9$，非常的显然，所以如果我们把 $1 \sim 9$ 的数字全部都用 $114514$ 表示出来的话，那么任意数字小于 $1e10$ 的数字就可以恶臭的论证了。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
#define in read()

inline int read(){
	int x = 0; char c = getchar();
	while(c < '0' or c > '9') c = getchar();
	while('0' <= c and c <= '9')
		x = x * 10 + c - '0', c = getchar();
	return x;
}

int num = 0;
map<int, string> m;

int main(){
	m[0] = "(1 - 1) * 4514";
	m[1] = "11 / (45 - 1) * 4";
	m[2] = "-11 + 4 - 5 + 14";
	m[3] = "11 * (-4) + 51 - 4";
	m[4] = "-11 - 4 + 5 + 14";
	m[5] = "11 - 4 * 5 + 14";
	m[6] = "1 - 14 + 5 + 14";
	m[7] = "11 - 4 + 5 - 1 - 4";
	m[8] = "11 - 4 + 5 / 1 - 4";
	m[9] = "11 - 4 + 5 + 1 - 4";
	m[10] = "-11 / 4 + 51 / 4";
	while(1){
		num = in;
		if(num <= 10) { cout << m[num] << '\n'; continue; }
		int d = num % 10; num /= 10;
		string ans = m[d];
		int k = 0;
		while(num){
			k++;
			d = num % 10; num /= 10;
			if(d == 0) continue;
			ans += " + ((" + m[d] + ")" + " * " + "(" + m[10] + ")" + " ^ " + "(" + m[k] + ")" + ")";
		}
		cout << ans << '\n';
	}
	return 0;
} 
```
