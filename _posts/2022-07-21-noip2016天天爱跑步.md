# 天天爱跑酷

&emsp; 传送门：[天天爱跑步 NOIP 2016 T4](https://www.luogu.com.cn/problem/P1600)

## 题目大意

&emsp; 给你一棵树，有 $m$ 个玩家，每个玩家在 $0$ 时刻从 $s\_i$ 跑到 $t\_i$，速度都是每秒一条边。每个节点上有一个观察员，并且第 $i$ 个节点上的观察员会在第 $w\_j$ 秒观察这个节点。现在要求每个节点的观察员会观察到多少个玩家。

&emsp; 一个玩家被观察到当且仅当他在 $w\_i$ 秒时经过节点 $i$。

## 算法分析

&emsp; 我们考虑将一个玩家 $s \to t$ 的路径拆开，变成 $s \to lca(s, t) \to t$。分别考虑这两段路径上会不会被观察到。

1. 首先时 $s \to lca(s, t)$ 这一段。我们可以很容易想到，如果要在这一段被观察到，就必须满足这样一个式子： $dep\_s - dep\_i = w\_i$，其中 $dep\_i$ 表示节点 $i$ 在这棵树中的深度。我们稍微移一下项： $dep\_s = dep\_i + w\_i$。我们会发现右边这一坨对于每一个节点是一个定值，所以我们的任务就变成了，对于每一个节点 $i$ 找有多少个 $s$ 满足这个式子。
2. 然后 $lca(s, t) \to t$ 这一段也是同理。我们令 $dis(x, y) = dep\_x + dep\_y - 2dep\_{lca(x, y)}$，那么这一段里面就是满足： $dep\_i + dis(s, t) - dep\_t = w\_i$，移一下项：$dep\_t - dis(s, t) = dep\_i - w\_i$。这需要注意因为不保证 $dep\_i > w\_i$，所以我们实际操作的时候应该把等式左右两边都加上 $n$ 来统计。

&emsp; 我们以的 $s \to lca(s, t)$ 为例，我们要找的就是在 $i$ 的子树中，有多少节点满足 $dep\_s = w\_i + dep\_i$。那么问题就能转化成对于一个 $s$，我们在从 $s \to lca$ 的路径上的所有点上全都放上一个类型为 $dep\_s$ 的物品，然后对于点 $i$ 我们要查找类型为 $dep\_i + w\_i$ 的物品的个数。这样我们很容易就能想到树上差分。

&emsp; 具体来说就是每个节点上放一个桶，然后对于每一个路径在桶里面操作差分就好了。最后统计就直接算子树和。但是这样的空间复杂度都有点爆炸，是 $O(n^2)$ 的。所以我们考虑优化这个做法。

&emsp; 既然不能每个点都开一个桶，我们就考虑全局开桶，具体来说，我们对树上的每一个节点开一个 $vector$，存储差分的时候我们进行的修改。然后我们全局开一个桶（其实是两个，因为有两种路径）。然后我们做一遍 $dfs$。

&emsp; 在 $dfs$ 到点 $x$ 的时候我们记录 $cnt$ 表示当前还没有继续进入子树的时候 $dep\_x + w\_x$ 类型物品的个数，然后我们再处理这个点对于桶的贡献（就是书上差分求和），最后再继续向下 $dfs$。再一次从子树回到这个点的时候，我们就能统计这个点的答案就是现在的 $dep\_x + w\_x$ 的个数减去前面我们记录的没有进入子树时的个数。

&emsp; 完结撒花！！！


## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
#define in read()
#define MAXN 300300
#define MAXM MAXN << 2

inline int read(){
	int x = 0; char c = getchar();
	while(c < '0' or c > '9') c = getchar();
	while('0' <= c and c <= '9'){
		x = x * 10 + c - '0'; c = getchar();
	}
	return x;
}

int w[MAXN] = { 0 };
int n = 0; int m = 0;
struct Tnode{
	int opt, val, c;                     // 路径种类, 值, 差分 
	Tnode(int o = 0, int v = 0, int C = 0) { opt = o, val = v, c = C; }
};
vector<Tnode> num[MAXN];
int tot = 0;
int first[MAXN] = { 0 };
int   nxt[MAXM] = { 0 };
int    to[MAXM] = { 0 }; 
int   dep[MAXN] = { 0 };
inline void add(int x, int y){
	nxt[++tot] = first[x];
	first[x] = tot; to[tot] = y;
}

int f[MAXN][50];
void prework(int x, int fa){
	dep[x] = dep[fa] + 1;
	for(int i = 0; i <= 30; i++) f[x][i + 1] = f[f[x][i]][i];
	for(int e = first[x]; e; e = nxt[e]){
		int y = to[e];
		if(y == fa) continue;
		f[y][0] = x; prework(y, x);
	}
}

int Lca(int x, int y){
	if(dep[x] < dep[y]) swap(x, y);
	for(int i = 30; i >= 0; i--){
		if(dep[f[x][i]] >= dep[y]) x = f[x][i];
		if(x == y) return x;
	}
	for(int i = 30; i >= 0; i--)
		if(f[x][i] != f[y][i]) x = f[x][i], y = f[y][i];
	return f[x][0];
}

int up[MAXN] = { 0 };                                 // s -> lca(s, t) 的路径 
int down[MAXN] = { 0 };                               // lca(s, t) -> t
int ans[MAXN] = { 0 };
void dfs(int x, int fa){
	int cnt = up[dep[x] + w[x]] + down[dep[x] - w[x] + n];
	for(int i = 0; i < num[x].size(); i++){                 // 差分子树和 
		Tnode t = num[x][i];
		if(t.opt == 1) up[t.val] += t.c;
		if(t.opt == 2) down[t.val] += t.c;
	}
	for(int e = first[x]; e; e = nxt[e]){
		int y = to[e];
		if(y == fa) continue;
		dfs(y, x);
	}
	ans[x] = up[dep[x] + w[x]] + down[dep[x] - w[x] + n] - cnt;
}

int main(){
	n = in; m = in;
	for(int i = 1; i < n; i++){
		int x = in, y = in;
		add(x, y), add(y, x);
	} prework(1, 0);
	for(int i = 1; i <= n; i++) w[i] = in;
	for(int i = 1; i <= m; i++){
		int s = in, t = in;
		int lca = Lca(s, t), dis = dep[s] + dep[t] - 2 * dep[lca];
		num[s].push_back(Tnode(1, dep[s], 1));
		num[lca].push_back(Tnode(1, dep[s], -1));
		num[t].push_back(Tnode(2, dep[t] - dis + n, 1));
		num[f[lca][0]].push_back(Tnode(2, dep[t] - dis + n, -1));
	} dfs(1, 0);
	for(int i = 1; i <= n; i++) cout << ans[i] << ' '; puts("");
	return 0;
}
```
