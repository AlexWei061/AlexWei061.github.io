# Splay

&emsp; 上次写的 $splay$ 只是把关键的 $rotate$ 和 $splay$ 函数大概讲了一遍，但是没有写剩下的操作。所以在这里，我们把剩下的操作补完吧 $qwq$。

&emsp; 上一篇传送门： [Splay](/posts/splay1/)

## 最开始的结构体

```cpp
struct Tnode{
    int fa, ch[2];
    int val, cnt, son;
}t[MAXN];
```

&emsp; $fa$ 就是父节点，$ch[0]$ 和 $ch[1]$ 分别表示左儿子和右儿子。 $val$ 表示这个节点的值，$cnt$ 表示数字的个数，$son$ 表示子树大小。

## 操作们

### get & up

&emsp; $get(x)$ 用来判断这个节点是他的父亲的左儿子或是右儿子。

&emsp; $up(x)$ 用来上传信息。

```cpp
inline bool get(int x) { return x == rs(t[x].fa); }
inline void up(int x) { t[x].son = t[ls(x)].son + t[rs(x)].son + t[x].cnt; }
```

### rotate & splay

&emsp; 首先就是上一篇文章里面介绍的两个操作了：

```cpp
void rot(int x){
    int y = t[x].fa, z = t[y].fa, k = get(x);
    t[z].ch[get(y)] = x; t[x].fa = z;
    t[y].ch[k] = t[x].ch[k ^ 1]; t[t[x].ch[k ^ 1]].fa = y;
    t[x].ch[k ^ 1] = y; t[y].fa = x;
    up(y); up(x);
}

void spl(int x, int to){
    while(t[x].fa != to){
        int y = t[x].fa, z = t[y].fa;
        if(z != to) rot(get(x) ^ get(y) ? x : y);
        rot(x);
    } if(!to) root = x;
}
```

### insert

&emsp; 这个操作用于插入一个新的数进入原本的平衡树。

&emsp; 就是顺着平衡树一直往下找，直到不存在这个数字或者找到这个数字所在的节点。

&emsp; 如果存在这个数字，那就直接把这个节点的 $cnt++$ 就好了。

&emsp; 如果不存在那就新建一个节点，加到原本的树上。并将它转到根。

```cpp
void ins(int x){
    int p = root, fa = 0;
    while(p and t[p].val != x) fa = p, p = t[p].ch[x > t[p].val];            // 一直向下找，比它小就走左边，比他大就走右边
    if(p) t[p].cnt++;
    else{
        p = ++tot;
        if(fa) t[fa].ch[x > t[fa].val] = p;
        t[p].fa = fa; t[p].val = x;
        t[p].cnt = 1; t[p].son = 1;
        t[p].ch[0] = t[p].ch[1] = 0;
    }
    spl(p, 0);
}
```

### find

&emsp; 这个操作用于找到一个数字的位置。并把这个位置转到根。

&emsp; 这个操作主要为了后面的操作服务，而且实现起来非常简单，就不多说了：

```cpp
void find(int x){
    int p = root;
    if(!p) return;                   // 不存在节点，无法找排名
    while(t[p].ch[x > t[p].val] and x != t[p].val) p = t[p].ch[x > t[p].val];          // 和 ins 中一样 一直向下找
    spl(p, 0);
}
```

### prefix & suffix

&emsp; 用来找前驱或者后继。

&emsp; 因为有了刚才的 $find(x)$ 的操作，这两个操作就会变得十分简单。我们以找前驱为例。

&emsp; 首先 $find(x)$	就把 $x$ 所在的点转到根了。当然也可能找不到 $x$。

&emsp; 如果找不到 $x$ ，那么就判断一下 $x$ 和当前这个点的大小关系，考虑输出。

&emsp; 如果找到 $x$，那么显然 $x$ 的前驱就是他左儿子的子树中最右边的那个点。所以我们一直往下找就可以了。

&emsp; 当然，我们还可以写得更简便一些，把 $pre$ 和 $suf$ 两个函数合并成一个函数一起写。

```cpp
int nxt(int x, int f){
    find(x);
    int p = root;
    if((t[p].val > x and f) or (t[p].val < x and !f)) return p;
    p = t[p].ch[f];
    while(t[p].ch[f ^ 1]) p = t[p].ch[f ^ 1];
    return p;
}
```

### delete

&emsp; 这个操作用于删除已经插入的某一个数。

&emsp; 我们先找到这个数 $x$ 的前驱和后继。然后把前驱 $splay$ 到根，然后把后继 $splay$ 到前驱。那么根据平衡树的性质，现在的 $x$ 一定是后继的左儿子，而且后继的左儿子一定没有子树。

&emsp; 如果这个数有很多个那就直接删一个（就是 $cnt--$） 。如果只有一个那么就直接让后继的左儿子为 $0$ 就好了（因为它没有子树）。

```cpp
void del(int x){
    int la = nxt(x, 0), nx = nxt(x, 1);                    // 前驱和后继
    spl(la, 0); spl(nx, la); int d = ls(nx);
    if(t[d].cnt > 1) t[d].cnt--, spl(d, 0);
    else t[nx].ch[0] = 0;
}
```

### rank

&emsp; 这个操作就是按排名查值了。

&emsp; 我们也是从根节点往下一直找。只是条件变了一下。

&emsp; 如果 $k$ 大于这个点加上左子树的大小，那么就应该往右走，并且排名减去这个点的大小加上左子树的大小的和。

&emsp; 如果不成立，还要分两种情况。第一种，如果排名小于等于当前节点子树大小，那就往左走。第二种，如果大于的话就直接返回这个数字。

```cpp
int rank_k(int k){
    int p = root;
    if(t[p].son < k) return false;
    while(114514){
        if(k > t[ls(p)].son + t[p].cnt) k -= t[ls(p)].son + t[p].cnt, p = rs(p);
        else
            if(t[ls(p)].son >= k) p = ls(p);
            else return t[p].val;
    }
}
```

## 完整代码

&emsp; 完整的代码贴在下面了。

&emsp; 题目传送门：[P3369 普通平衡树](https://www.luogu.com.cn/problem/P3369)

```cpp
#include<bits/stdc++.h>
using namespace std;
#define in read()
#define MAXN 100100
#define ls(p) t[p].ch[0]
#define rs(p) t[p].ch[1]
#define INFI 1 << 30

inline int read(){
    int x = 0; int f = 1; char c = getchar();
    while(c < '0' or c > '9'){
        if(c == '-') f = -1; c = getchar();
    }
    while('0' <= c and c <= '9'){
        x = x * 10 + c - '0'; c = getchar();
    }
    return f * x;
}

int n = 0;
int tot = 0, root = 0;
struct Tnode{
    int fa, ch[2];
    int val, cnt, son;
}t[MAXN];

inline bool get(int x) { return x == rs(t[x].fa); }
inline void up(int x) { t[x].son = t[ls(x)].son + t[rs(x)].son + t[x].cnt; }

void rot(int x){
    int y = t[x].fa, z = t[y].fa, k = get(x);
    t[z].ch[get(y)] = x; t[x].fa = z;
    t[y].ch[k] = t[x].ch[k ^ 1]; t[t[x].ch[k ^ 1]].fa = y;
    t[x].ch[k ^ 1] = y; t[y].fa = x;
    up(y); up(x);
}

void spl(int x, int to){
    while(t[x].fa != to){
        int y = t[x].fa, z = t[y].fa;
        if(z != to) rot(get(x) ^ get(y) ? x : y);
        rot(x);
    } if(!to) root = x;
}

void ins(int x){
    int p = root, fa = 0;
    while(p and t[p].val != x) fa = p, p = t[p].ch[x > t[p].val];            // 一直向下找，比它小就走左边，比他大就走右边
    if(p) t[p].cnt++;
    else{
        p = ++tot;
        if(fa) t[fa].ch[x > t[fa].val] = p;
        t[p].fa = fa; t[p].val = x;
        t[p].cnt = 1; t[p].son = 1;
        t[p].ch[0] = t[p].ch[1] = 0;
    }
    spl(p, 0);
}

void find(int x){
    int p = root;
    if(!p) return;                   // 不存在节点，无法找排名
    while(t[p].ch[x > t[p].val] and x != t[p].val) p = t[p].ch[x > t[p].val];          // 和 ins 中一样 一直向下找
    spl(p, 0);
}

int nxt(int x, int f){
    find(x);
    int p = root;
    if((t[p].val > x and f) or (t[p].val < x and !f)) return p;
    p = t[p].ch[f];
    while(t[p].ch[f ^ 1]) p = t[p].ch[f ^ 1];
    return p;
}

void del(int x){
    int la = nxt(x, 0), nx = nxt(x, 1);                    // 前驱和后继
    spl(la, 0); spl(nx, la); int d = ls(nx);
    if(t[d].cnt > 1) t[d].cnt--, spl(d, 0);
    else t[nx].ch[0] = 0;
}

int rank_k(int k){
    int p = root;
    if(t[p].son < k) return false;
    while(114514){
        if(k > t[ls(p)].son + t[p].cnt) k -= t[ls(p)].son + t[p].cnt, p = rs(p);
        else
            if(t[ls(p)].son >= k) p = ls(p);
            else return t[p].val;
    }
}

int main(){
    ins(INFI); ins(-INFI);
    n = in;
    while(n--){
        int op = in;
        if(op == 1) ins(in);
        if(op == 2) del(in);
        if(op == 3) find(in), cout << t[ls(root)].son << '\n';
        if(op == 4) cout << rank_k(in + 1) << '\n';
        if(op == 5) cout << t[nxt(in, 0)].val << '\n';
        if(op == 6) cout << t[nxt(in, 1)].val << '\n';
    }
    return 0;
}
```
