# Alex's Blog

A lightweight Jekyll technical blog hosted on GitHub Pages.

## Writing

Add new articles under `_posts/` with this front matter:

```yaml
---
layout: post
title: Article Title
date: YYYY-MM-DD
tags: [tag-one, tag-two]
summary: Short article summary.
math: false
---
```

Use `math: true` for posts that need MathJax rendering.![img](./static/media/icon.06a6aa23.png)
